# 百度搜索页面演示项目

## 项目介绍
这是一个使用 Vue 3 和 Vite 构建的百度搜索页面演示应用，实现了百度搜索主页的基本功能和样式，包括搜索框、联想搜索下拉列表等特性。

## 功能特性

- **搜索框功能**: 输入关键词进行搜索
- **联想搜索**: 根据输入内容实时显示匹配的搜索建议 <mcfile name="src/App.vue" path="c:\Users\洛伦兹力不做功\baidu-demo\src\App.vue"></mcfile>
- **下拉菜单**: 聚焦时显示搜索建议列表
- **响应式设计**: 页面居中布局，适配不同屏幕尺寸
- **搜索按钮**: 点击后执行搜索操作
